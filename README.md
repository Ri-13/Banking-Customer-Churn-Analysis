🏦 Banking Customer Churn Analysis (EDA Project)

📌 Project Overview
This project explores banking customer data to analyze churn patterns using Exploratory Data Analysis (EDA), data cleaning, and visualization techniques. The objective is to identify trends and key factors associated with customer attrition to support data-driven retention strategies.

🎯 Business Objective
1.Customer churn significantly impacts profitability. 

2.Measure overall churn rate

3.Identify customer segments with higher churn

4.Discover patterns influencing customer exit behavior

📊 Analysis & Key Metrics

1.Overall Churn Rate

2.Churn by Gender

3.Churn by Geography

4.Churn by Credit Card Possession and Credit Scores.

5.Churn by Number of Products

6.Churn by Tenure Distribution


7.Account Balance & Credit Score Analysis
8.Active vs Inactive Members Comparison


🛠 Tools & Technologies Used
1.Python (Pandas, NumPy, Matplotlib/Seaborn)
2.SQL (for querying and data extraction)
